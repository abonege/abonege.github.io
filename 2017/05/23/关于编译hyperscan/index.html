关于编译hyperscan

CMAKE

hyperscan是基于cmake生成Makefile，所以我们先需要下载cmake，我这里用的是3.9.5版本。
cmake我理解就是一个自动化构建编译工程文件(比如makefile，VS的 .proc文件)的东西,类似bjam。

至此cmake安装完毕

安装依赖

boost

需要boost的regex lib，我们把boost解压到/opt目录下

ragel

我安装的是6.10版本

pcap

我安装的是1.8.1版本

dbus

我安装的是dbus-1.11.12版本
